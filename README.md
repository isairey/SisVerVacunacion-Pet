# 🐾💉 Sistema de Control de Vacunación de Mascotas

Aplicación web desarrollada para gestionar el control de vacunas en mascotas, permitiendo llevar un registro organizado de aplicaciones, fechas y próximas dosis.

---

## 🚀 Descripción

Este sistema facilita el seguimiento del historial de vacunación de mascotas, ayudando a veterinarios y dueños a mantener al día las vacunas necesarias.

Permite:

- Registro de mascotas 🐶🐱  
- Control de vacunas aplicadas 💉  
- Programación de próximas dosis 📅  
- Consulta de historial clínico 🩺

## ⚙️ Instalación

### 1. Clonar el repositorio

```bash
git clone https://github.com/isairey/SisVerVacunacion-Pet.git
```
2. Acceder al proyecto
```
cd sistema-vacunacion-mascotas
```
3. Crear entorno virtual (opcional)
```
python -m venv venv
source venv/bin/activate   # Linux / Mac
venv\Scripts\activate      # Windows
```
4. Instalar dependencias
```
pip install -r requirements.txt
```
5. Ejecutar el servidor
```
python app.py
```
## 🌐 Uso

Abrir en el navegador:
```
http://localhost:5000
```
